3. Использование изображений на странице.
HTML: Используйте тег img для вставки изображения прямо в содержимое страницы.
Укажите атрибут src для указания пути к изображению.
Используйте атрибут alt для добавления текстового описания изображения (важно для доступности).
CSS: Вы можете задать изображение как фон для элемента с помощью свойства background-image.
Указать размеры изображения можно атрибутами width и height в теге img или свойства width и height в CSS.
