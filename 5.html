5. Фреймы, плавающие фреймы, формы
Фреймы (Frames):
Фреймы позволяют разбить окно браузера на несколько частей, каждая из которых может загружать отдельную HTML-страницу.
Обычно фреймы определяются с помощью элемента frame или iframe.
Плавающие фреймы (Floating Frames):
Плавающие фреймы похожи на обычные фреймы, но они могут быть перемещены пользователем по экрану.
Они создаются с помощью элемента iframe и использования JavaScript для управления их перемещением.
Формы (Forms):
Формы HTML позволяют пользователям взаимодействовать с веб-страницей, отправляя данные на сервер.
Форма создается с помощью элемента form, внутри которого содержатся элементы ввода (такие как текстовые поля, кнопки, флажки и т. д.).
После заполнения формы и отправки пользователем, данные обычно обрабатываются на сервере с помощью серверного скрипта (например, PHP, Python, Ruby).
