1. Язык разметки HTML. Синтаксис HTML
HTML (HyperText Markup Language) - это язык разметки для создания веб-страниц.
Он состоит из элементов, которые определяют структуру и содержание страницы.
Синтаксис HTML включает в себя использование тегов для определения элементов и их атрибутов для уточнения поведения.
Основные компоненты синтаксиса HTML:
!DOCTYPE html: Определение типа документа.
html: Начало и конец HTML документа.
head: Содержит метаинформацию о странице.
title: Определяет заголовок страницы в браузере.
body: Определяет тело документа.
h1, p, a, img: Примеры различных элементов (заголовок, абзац, ссылка, изображение).
